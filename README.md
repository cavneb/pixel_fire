## Installation
Add this line to your application's Gemfile:

```ruby
gem 'pixel_fire'
```

And then execute:
```bash
$ bundle
$ bin/rails pixel_fire:install:migrations
$ bin/rails db:migrate
```

Add the engine to your routes file:
```ruby
mount ::PixelFire::Engine, at: "/pixel_fire"
```
